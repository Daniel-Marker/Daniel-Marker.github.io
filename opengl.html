OpenGL Project

What is it?

This project is a simple game made using OpenGL, freeglut and glew.

Who did what?

During tutorials, we were given some code which could be used to make a basic OpenGL application. However, this was for OpenGL 1.1. So, I ported the tutorial code to modern OpenGL and updated it based off of code from The Cherno's OpenGL series (1-27). Other than using these as a base, everything else on the project is made by myself.

When was it worked on?

Work started on the project around January of 2020. The project will be worked on until May 1st 2020

Where can I play the game/access the source code?

As the project is still in development, the source code and game are currently not available. However, once the deadline has passed a link to the repo will be made available.

Where can I stay updated on the project?

I post regularly about this OpenGL project on my twitter under the hashtag #opengl
