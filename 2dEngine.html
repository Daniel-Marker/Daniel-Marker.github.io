2D Engine

What is it?

This project is a 2D tile based engine made using C++, OpenGL 4.5, glew and glfw.
For this, my goal is to create an engine that can draw a layer of tiles with a single render call. This is possible as the tiles in a given layer all share the same texture. For this project, I have been heavily inspired by my work for the Gameboy, as the graphics hardware is relatively easy to interface with. Additionally, there are certain effects that older hardware lends itself to that aren't as easy on modern hardware. For example, simple parallax scrolling was typically implemented by setting the x and y offset of the background per scanline.

When did work start?

Work started on the project around September of 2020. I am currently aiming to finish development around the summer of 2021.

Where can I access the source code?

Currently, the project is still early in development, so the source code is not public. However, once more work has been done on it, the repo will be made public and will be linked here.
